Contact Center Demos

This repository contains my Amazon Connect demo suite, including:

- Contact flows for inbound, outbound, self-service, and callbacks
- Multi-language support (English, Spanish, French, more)
- SMS-first and voice-first workflows
- QA and call scoring using Contact Lens APIs
- Real-time analytics and post-call summaries
- WFM demo: forecasting, adherence, and performance insights
- AI-driven agent assist via Amazon Q or Bedrock
- Integration with my AWS Knowledge API
- Salesforce CTI concepts (for future demos)
